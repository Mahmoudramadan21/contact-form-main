# Contact Form Project

This is a responsive and accessible contact form built using HTML, CSS, and JavaScript. The form includes validation for all fields and provides feedback to the user upon successful submission.

## Features

- **Responsive Design**: Works on all screen sizes.
- **Accessibility**: Built with accessibility in mind, ensuring it is usable by everyone.
- **Form Validation**: Validates all fields before submission.
- **Success Message**: Displays a success message upon successful form submission.

## Technologies Used

- **HTML**: For structuring the form.
- **CSS**: For styling the form and making it responsive.
- **JavaScript**: For form validation and handling user interactions.

## How to Use

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Mahmoudramadan21/6.-contact-form-main.git
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd 6.-contact-form-main
   ```
3. **Open the Project**:

   Open the `index.html` file in your browser to view the form.

## Project Structure

- `index.html`: The main HTML file containing the form structure.
- `css/style.css`: The CSS file for styling the form.
- `js/main.js`: The JavaScript file for form validation and interaction handling.
- `assets/`: Directory containing any assets like images or icons.
